Автоматизация загрузки ВКР и учебных практик

  • Для доступа к API необходим ключ доступа - SECRET_KEY_THESIS: {{thesis_key}}

  • Скачайте Python скрипт для загрузки работ

  • Настройте скрипт под загружаемую работу (название, ФИО автора, год и т.д.)

  • Запустите скрипт

  • В случае успеха скрипт вернет "{"status":0,"string":"Success"}"

  • Максимальный размер загружаемого файла 70 Мб (в /etc/nginx/sites-enabled/se.math.spbu.ru стоит client_max_body_size 70M;)

  • В случае ошибки вернет причину ошибки, например: {"status":500,"string":"Invalid secret key: fbe3a36f1fb3eff7aeeea5f18865c8f2"}

  • Все вновь загруженные работы помечаются как временные и доступны по адресу https://se.math.spbu.ru/theses_tmp.html

  • Проверьте вновь загруженные работы и если все правильно, подтвердите их и они станут доступны в общем архиве

  • По всем вопросам и пожеланиям пишите мне в телеграм @Zelenchuk
